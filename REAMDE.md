# 📄 Template de Currículo em LaTeX — Otimizado para ATS

Template profissional em LaTeX com guia completo de preenchimento,
pensado para passar pelos sistemas de triagem automática (ATS)
usados por empresas de tecnologia.

## O que é ATS?

Applicant Tracking System — software usado por recrutadores para
filtrar currículos automaticamente antes de qualquer humano ler.
Currículos mal estruturados são eliminados antes de chegar ao RH.

## O que este template resolve

- Estrutura 100% legível por ATS (sem tabelas, caixas ou ícones)
- Comentários em cada seção explicando o que escrever e por quê
- Dicas de palavras-chave e formatação para cada bloco
- Checklist final antes de enviar

## Como usar

1. Clone o repositório
2. Abra o `template.tex` em qualquer editor LaTeX
   (Overleaf, VS Code + LaTeX Workshop, TeX Studio)
3. Siga os comentários de cada seção
4. Compile para PDF e envie

## Compilar online (sem instalar nada)

Acesse [overleaf.com](https://overleaf.com), crie um projeto
e cole o conteúdo do `template.tex`.
