markdown# 📄 Currículo LaTeX — Template Otimizado para ATS

> Template profissional em LaTeX com guia completo de preenchimento,
> desenvolvido para maximizar a aprovação em sistemas de triagem
> automática (ATS) utilizados por empresas de tecnologia.

---

## 🧩 O problema que este projeto resolve

A maioria dos currículos é eliminada **antes de qualquer humano ler**.

Empresas de médio e grande porte utilizam sistemas chamados **ATS
(Applicant Tracking System)** para filtrar automaticamente os
candidatos com base em palavras-chave, estrutura do documento e
formatação. Currículos com tabelas, caixas de texto, ícones ou
estrutura inadequada simplesmente não passam pelo filtro.

Este template foi construído para resolver exatamente isso.

---

## ✅ O que este template oferece

- Estrutura 100% legível por ATS — sem tabelas, caixas ou elementos gráficos
- Comentários detalhados em **cada seção** explicando o que escrever e por quê
- Guia de palavras-chave: como espelhar a linguagem da vaga no currículo
- Fórmula CAR aplicada à experiência profissional (Contexto → Ação → Resultado)
- Checklist final de validação antes do envio
- Layout limpo e profissional gerado diretamente em PDF via LaTeX

## 🚀 Como usar

### Opção 1 — Overleaf (recomendado, sem instalar nada)

1. Acesse [overleaf.com](https://overleaf.com) e crie uma conta gratuita
2. Clique em **New Project → Blank Project**
3. Cole o conteúdo do arquivo `template.tex`
4. Siga os comentários de cada seção para preencher
5. Compile e baixe o PDF

### Opção 2 — Localmente

```bash
# Clone o repositório
git clone https://github.com/luisFernandoJv/curriculo-latex-template-ats.git
cd curriculo-latex-template-ats

# Compile com pdflatex
pdflatex template.tex
```

> **Requisito:** TeX Live, MiKTeX ou qualquer distribuição LaTeX instalada.
> Para Windows, recomenda-se o [MiKTeX](https://miktex.org/).
> Para Linux: `sudo apt install texlive-full`

## 📋 Seções do template

| Seção                        | O que contém                                       |
| ---------------------------- | -------------------------------------------------- |
| **Cabeçalho**                | Nome, título profissional, contatos e links        |
| **Resumo Profissional**      | Apresentação objetiva com palavras-chave da vaga   |
| **Competências Técnicas**    | Stack organizada por categoria para leitura do ATS |
| **Experiência Profissional** | Bullets com fórmula CAR e verbos de ação           |
| **Projetos Relevantes**      | Projetos reais com stack, contexto e resultado     |
| **Formação Acadêmica**       | Grau, curso e instituição no formato correto       |
| **Cursos e Certificações**   | Certificações com nome oficial e emissor           |
| **Checklist Final**          | Validação completa antes de enviar o currículo     |

## 🧠 Conceitos abordados nos comentários

- **O que é ATS** e como ele filtra currículos automaticamente
- **Palavras-chave**: como identificá-las na vaga e onde inserir no currículo
- **Fórmula CAR** (Contexto, Ação, Resultado) para descrever experiências
- **Verbos de ação**: lista de verbos recomendados por tipo de entrega
- **Erros comuns** que eliminam currículos bons nos filtros automáticos
- **Formatação segura**: o que o ATS lê e o que ele ignora
